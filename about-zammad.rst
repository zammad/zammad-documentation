Zammad
******

Do you receive many emails and want to answer them with a team of agents?

You're going to love Zammad_!

Zammad is a web based open source helpdesk/customer support system with many features to manage customer communication via several channels like telephone, facebook, twitter, chat and emails. It is distributed under the GNU AFFERO General Public License (AGPL).

The code is open source, and `available on GitHub`_!

.. _available on GitHub: https://github.com/zammad/zammad
